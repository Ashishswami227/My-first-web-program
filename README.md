# Personal Portfolio Website (Full-Stack)

This is a full-stack personal portfolio website built to showcase projects and web development skills.

## Tech Stack Used:
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose Object Modeling)

## Key Features Implemented:
- Responsive user portfolio page.
- RESTful API built with Express.js to fetch data from database.
